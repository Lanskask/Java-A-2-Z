# Tracker program
Program to track items

Now using Postgres.

## Task has
- id
- name
- description
- creation date (Date type)
- Comments List (String array)

## Functions to operate
- Add new task
- Edit task
- Delete task
- Show all tasks
- Show filtered list


## SQL Tracker functions:
- add
- update
- delete
- findAll
- findByName

## SQL Methods to make queries
- executeUpdate for:
    - INSERT
    - UPDATE
    - DELETE
    - для команд определения данных:
        - CREATE TABLE
        - DROP TABLE
- executeQuery for
    - SELECT
- execute
    - для интерактивного создания запросов
